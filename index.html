Bitcoinvert---VirWox---Android by Deathnetworks

Bitcoin currency convertor based around VirWox

currency conversion for bitcoin to standard currency via the VirWox.com system, this system works based on the usage of the market order system include the fees that are required to perform these actions. amounts can only be traded for and against solid amounts such as 1Bitcoin as apposed to 0.01Bitcoin or £10 as opposed to £10.50. it also allows the user to switch between 3 currency types USD/EURO/GBP - pick to include the fee's that virwox charges or not - select to include the paypal withdrawal charge in the conversion or not

Install

Download apk use a file explorer to select the file, install and run as normal.
